Granite State Precision Pole Inspection, LLC Website

How to publish on GitHub Pages:

1. Unzip this package.
2. In your GitHub repository, click Add file > Upload files.
3. Upload ALL unzipped contents:
   - index.html
   - styles.css
   - images folder
4. Click Commit changes.
5. In GitHub, go to Settings > Pages.
6. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
7. Save.
8. Wait 1-3 minutes for GitHub Pages to publish.

Your website URL will usually be:
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/

If your repo is named granite-state-pole-inspection, then it will be:
https://YOUR-USERNAME.github.io/granite-state-pole-inspection/
